Part IV

Configuring and Monitoring Deployment

The topics in this part provide information on the Java Control Panel, deployment properties, and the Java Console. The Java Control Panel and deployment properties are used to configure deployment options. The Java Console is used to monitor applications with traces and logging information.

Part IV contains the following topics:

  • Chapter 20, "Java Control Panel"

  • Chapter 21, "Deployment Configuration File and Properties"

  • Chapter 22, "Java Console, Tracing, and Logging"
