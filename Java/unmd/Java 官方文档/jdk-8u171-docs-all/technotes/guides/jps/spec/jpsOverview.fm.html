CHAPTER 1

Introduction




The Java Print Service (JPS) is a Java Print API that is designed to support printing on all Java platforms, including platforms requiring a small footprint, but also supports the current Java Print API. This unified Java Print API includes extensible print attributes based on the standard attributes specified in the Internet Printing Protocol (IPP) 1.1 from the IETF Specification, RFC 2911. With the attributes, client and server applications can discover and select printers that have the capabilities specified by the attributes. In addition to the included StreamPrintService, which allows applications to transcode print data to different formats, a third party can dynamically install their own print services through the Service Provider Interface.
