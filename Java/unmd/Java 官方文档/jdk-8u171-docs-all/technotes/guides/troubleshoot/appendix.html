Appendix

    Describes environment variables and system properties that are useful in troubleshooting issues with Java HotSpot Server VM.

    Describes command-line options that are useful in diagnosing issues with Java HotSpot Server VM.

    Provides a summary of tools available in the current and previous release of JDK.
