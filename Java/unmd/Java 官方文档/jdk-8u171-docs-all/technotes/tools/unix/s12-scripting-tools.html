16 Scripting

Use this command to run scripts that interact with the Java platform. This command is unsupported and experimental and might not be available in future JDK releases.

This part contains the following command:

jjs: Runs the Nashorn command-line script shell.

jrunscript: Experimental. Runs a command-line script shell that supports interactive and batch modes.
