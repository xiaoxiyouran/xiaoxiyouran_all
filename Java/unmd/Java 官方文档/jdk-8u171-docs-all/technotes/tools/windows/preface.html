Preface

The Java Platform, Standard Edition (Java SE) Command Reference describes the valid options and arguments for Java SE commands. In many cases, examples are included to show correct usage. If you want conceptual information or need step-by-step instructions about these topics, see The Java Tutorials at
http://docs.oracle.com/javase/tutorial/index.html

Audience

This document is intended for Java SE developers who want to use the commands provided in the JDK.
