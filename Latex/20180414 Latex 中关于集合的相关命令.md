# 20180414 Latex 中关于集合的相关命令



------

原文链接： https://blog.csdn.net/hunauchenym/article/details/9201371

需要的包
usepackage{amsmath,amssymb}

集合的大括号：                  \{ ...   }\

集合中的“|”：                \mid

属于：                        \in

不属于：                    \not\in

A包含于B：                A\subset B

A真包含于B：                A\subsetneqq B

A包含B：                    A\supset B

A真包含B：                A\supsetneqq B

A不包含于B：                A\not\subset B

A交B：                    A\cap B

A并B：                    A\cup B

A的闭包：                    \overline{A}

A减去B:                    A\setminus B

实数集合：                \mathbb{R}

空集：                    \emptyset
